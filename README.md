# hexo-style-scss

[![npm:](https://img.shields.io/npm/v/hexo-style-scss.svg?style=flat)](https://www.npmjs.com/packages/hexo-style-scss)

Transform SCSS code in `<style type="text/scss">...</style>` to CSS

## Installation

```bash
npm install --save hexo-style-scss
```
